Program 1b:Develop a program to read the name and year of birth of a person. Display whether the person is a senior citizen or not. in three subjects.

Description: This program demonstrates a basic implementation of an array traversal.

Code:

import datetime
name = input("Enter your name: ")
birth_year = int(input("Enter your year of birth: "))
current_time = datetime.datetime.now()
current_year = current_time.year
age = current_year - birth_year
if age>=60:
 print(name," is a senior citizen.")
else:
 print(name," is NOT a senior citizen.")
