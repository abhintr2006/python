Program 1:Develop a program to read the student details like Name, USN, and Marks in three subjects.

Description: This program demonstrates a basic implementation of an array traversal.

Code:

Name = input("Enter name of student: ")
USN = input("Enter USN of student: ")
print("Enter marks in three subjects: Physics,Chemistry and Maths : ")
marks_Phy = int(input("Enter marks in Physics: "))
marks_Che = int(input("Enter marks in Chemistry: "))
marks_Mat = int(input("Enter marks in Maths: "))
total_Marks = marks_Phy + marks_Che + marks_Mat
perct = total_Marks/3
print("The details of student are :")
print("Name: ",Name)
print("USN: ",USN)
print("Marks scored in 3 subjects: ")
print("Physics: ",marks_Phy)
print("Chemistry: ",marks_Che)
print("Maths: ",marks_Mat)
print("Total Marks : %3d, Percentage : %5.2f" % (total_Marks,perct))
